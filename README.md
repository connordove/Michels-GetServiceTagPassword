# Hello! Welcome to my first python program :)

## Please download [`Get LAPS.exe`](https://github.com/connordove/Michels-GetServiceTagPassword/blob/main/Get%20LAPS.exe)

### Steps to run:
1. Download `Get LAPS.exe`. You can view the programs code within the `main.pyw`.
2. Open `Get LAPS.exe`.
3. Enter the service number for the computer you'd like to get a password for.
4. The password will be displayed within the listbox below. If **ERROR** is displayed, it means LAPS did not provide a password. If | **mm/dd/yy** | that is the expiration date from LAPS. The password often works past the expiration date, but can be a contributing issue if the password is not working properly.
5. The QR code will be generated after searching a service number, or selecting one from the listbox. The QR automatically updates and displays the most recent password unless selecting a different password. The QR code can be scanned with a handheld QR scanner to speed up the login process.
6. You can delete a selected password by selecting one from the listbox and pressing the `Delete Selected` button. This will only delete the selected password(s)
7. You can delete your entire password history by pressing the `Delete History` button. It will ask for confirmation to ensure you want to delete the entire history.
8. And that's it! Please let me know if you have any issues!!
